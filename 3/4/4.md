Let's Encrypt Zertifikat erstellen

Aufgabe: Generieren Sie ein "Let's Encrypt"-Zertifikat mit certbot.
Fragen:
Was ist ein "Let's Encrypt"-Zertifikat, und welche Organisation steht dahinter?
- Kostenloses SSL/TLS-Zertifikat
- Bereitgestellt von der Internet Security Research Group (ISRG)

Was ist essentiell dass Sie das Zertifikat anfragen können?
- Domainbesitz oder -verwaltung
- Zugriff auf den Server
- Certbot-Tool
- HTTP- oder DNS-Validierung

Wie lange ist ein solches Zertifikat gültig, und was passiert nach Ablauf?
- Gültigkeit: 90 Tage
- Nach Ablauf: Zertifikat wird ungültig, Verbindung unsicher
- Lösung: Automatische Erneuerung mit Cron-Job (Certbot)

Anleitung: Richten Sie auch ein automatisches Erneuerungssystem für das Zertifikat ein, z.B. durch einen Cron-Job, um sicherzustellen, dass die Zertifikate regelmäßig aktualisiert werden.


sudo certbot --apache -d sivs-dbd.freemyip.com

systemctl status certbot.timer
