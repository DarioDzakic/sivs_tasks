Firewall-Konfiguration

Aufgabe: Konfigurieren Sie die Firewall des Servers (ufw) so, dass nur die notwendigen Ports für HTTP, POSTGRES und SSH geöffnet sind.
Anleitung: Erforschen Sie, wie Sie ufw auf einem Ubuntu-Server nutzen können, um die Firewall entsprechend einzurichten. Stellen Sie sicher, dass alle anderen Ports blockiert sind, um unnötige Zugriffe zu verhindern.

sudo ufw allow .. (22/tcp)
sudo ufw enable
